## 概要

個人のブログとポートフォリオを公開するためのウェブサイトです。

## 特徴

- **高速**: Astro のゼロ JavaScript アプローチにより、高速なページ読み込みを実現
- **レスポンシブ**: Tailwind CSS によるモバイルファーストなデザイン
- **自動デプロイ**: GitHub Actions による CI/CD パイプライン

## 技術的なポイント

- Astro Content Collections による型安全なコンテンツ管理
- Tailwind CSS 4.x の新しい設定方式を採用
- GitHub Pages へのゼロコンフィグデプロイ
