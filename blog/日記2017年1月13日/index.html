コンソール上の警告([Vue warn]: Cannot find element) を削除する

参考URL: http://qiita.com/yoneapp/items/3fbe4c1c133cfef87230

自分のLaravel5.2 + Vue.js環境でも起きたので対処法をメモ.

document.addEventListener('DOMContentLoaded', function() {
  if (document.getElementById("feed")) { require('./feed.js') }
  if (document.getElementById("shop")) { require('./shop.js')  }
}, false);

Consider pre-initializing the property with the “data” option for more reliable reactivity and better performance.

参考URL: https://github.com/vuejs/Discussion/issues/352#issuecomment-132942027

For components, data must be function returning object:

data: function() {
  return {
    filterBy: 'none'
  }
}
